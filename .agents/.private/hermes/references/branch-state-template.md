# `states/<branch>/YYYYMMDD.md` template

このテンプレートは `.hermes-prj-states/states/<branch>/YYYYMMDD.md` 用。
その日、その branch で有効な事実だけを残す。
履歴を増やすためのログではなく、当日スナップショットとして更新する。
`STATE.md` の全文コピーは避け、branch 固有の差分と判断材料に絞る。
